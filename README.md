This is a simple and fast e-commerce web application built using React and Vite. It features a responsive landing page, category-specific product pages (like mobiles, laptops, etc.), and a functional cart system where users can add items. The app uses React Router for navigation and Context API for managing cart state. All assets are stored inside the public/assets folder and the UI is designed to be clean and user-friendly. Perfect for learning or showcasing a modern React project.
